About SpaceGuard

SpaceGuard is a simple JavaScript game that I've made during my holidays last Christmas, because I wanted to experiment with the animation capabilities of the new canvas element. This project is not finished yet, but it has reached a state where one can actually play it. Please read the instructions below.

Instructions

You can controll the guard (shield) with the mouse pointer. Random comets will approach the central starship from various angles of the map. Try to prevent the comets from reaching the starship and make new damages.

The guard can destroy the comets by touching them, but this will inflict some damage to it. Obtain power ups that occur randomly in order to proceed to new levels. You can defuse bombs by clicking (left-mouse-button) nearby a bomb.

The purpose of the game is to survive for a specific amount of time. Have fun!
